软件

Windows

HowieHye 自制

云萌Windows10激活工具

✅ 官网：https://cmwtat.cloudmoe.com

CloudMoe Windows 10 Activation Toolkit get digital license, the best open source Win 10 activator in GitHub. GitHub 上最棒的开源 Win10 数字权利（数字许可证）激活工具！

火绒安全

✅ 官网：https://www.huorong.cn/

PotPlayer

✅ 官网：https://potplayer.daum.net/

超强大的视频播放器

Everything

✅ 官网：https://www.voidtools.com/zh-cn/

电脑检索利器

Snipaste 截图工具

✅ 官网：https://www.snipaste.com/

IDM

✅ 官网：http://www.internetdownloadmanager.com/

Visual Studio Code

✅ 官网：https://code.visualstudio.com/download/

Geek

✅ 官网：https://geekuninstaller.com/

卸载神器，专注于卸载功能

Typora

✅ 官网：https://www.typora.io/

Markdown 编辑器

AdGuard

✅ 官网：https://adguard.com/zh_cn/adguard-browser-extension/chrome/overview.html

广告拦截工具，还有浏览器插件

坚果云收件箱

✅ 官网：https://workspace.jianguoyun.com/inbox/#/collect

疫情期间收健康码帮了我很大的忙

FinalShell

✅ 官网：https://www.hostbuf.com/

一体化的的服务器,网络管理软件,不仅是 ssh 客户端,还是功能强大的开发,运维工具,充分满足开发,运维需求。

TrayS

✅ 项目地址：https://gitee.com/cgbsmy/TrayS

WIN10 任务栏透明/调色/磨砂/亚克力

深蓝词库转换

✅ 项目地址：https://github.com/studyzy/imewlconverter

支持目前主流的各大输入法词库互相转换。

PowerToys

✅ 项目地址：https://github.com/microsoft/PowerToys

微软官方推出的一系列实用工具合集。简直就是官方外挂！

Mac OS

从 18 年到现在一直使用黑苹果，当然也有一些优质 APP 啦

ETCher

✅ 官网：https://www.balena.io/etcher/

简洁易用的镜像写入工具，三步做好启动盘。

当然也有 Windows 客户端，做黑苹果安装盘要用到

MOS

✅ 官网：https://mos.caldis.me/

平滑滚动小工具，让你的滚轮爽如触控板

iShot

✅ 官网：https://www.better365.cn/ishot.html

截图、贴图、录屏样样优秀

当然他家还有其他 APP

IINA

✅ 官网：https://iina.io/

美观简洁，功能强大，拿来看剧，爽的一批

ITerm2

✅ 官网：https://www.iterm2.com/

终端,功能强大,插件丰富

eul

✅ 官网：https://github.com/gao-sun/eul/

macOS status monitoring app

Android

Android 端的一些 App

Free

✅ 官网：http://app.free52.com/

很 Free 的刷剧 App

当然也有网页版

Alook

✅ 下载链接：https://www.coolapk.com/apk/alook.browser

iOS 第一浏览器发布安卓版
