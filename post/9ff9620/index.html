黑苹果安装指北手册(持续更新)

发表于2020-08-26|更新于2023-12-21|+ 黑苹果
|阅读量:

引言

最近发现158976808里还有很多小伙伴进群，但冰水加劲Q的黑苹果安装教程比较老旧了(由于冰水大佬使用的是 Clover 不再EFI仓库维护，而现在macOS Big Sur需要使用OC来引导)，所以就从黑苹果安装教程修改得到了这篇教程，美其名曰“指北手册”。

OC还是Clover？

引自Sukka：随着 OpenCore 的日渐成熟，Acidanthera团队宣布放弃旗下大部分内核驱动对 Clover 的兼容性支持，与其届时被迫换到 OpenCore ，不如主动开始迁移。

如果你使用的是 Clover ，你可以参考冰水加劲Q的黑苹果安装教程

安装

请注意备份数据！！！

来自黑果小兵的温馨提示：

准备

  • 至少8G容量的U盘
  • 镜像写入工具：Etcher
  • Disk Genius
  • 黑果小兵的 请关注微信公众号“黑果小兵的部落阁”
  • HowieHye的EFI
  • OpenCore Configurator
  • 冰水加劲Q的one-key-hidpi脚本

分配ESP分区

硬盘分区表的类型必须为GUID，ESP分区必须大于200M

分出你安装macOS的分区，大小自己决定，格式随意

写入镜像

  • 安装并打开Etcher
  • 选择镜像
  • 选择U盘
  • Flash，等待完成

替换EFI

下载HowieHye的EFI

请注意, 新安装用户请使用1ef8665以前的Commit

其他机型的小伙伴可以到黑果小兵整理的Hackintosh黑苹果长期维护机型EFI及安装教程整理寻找，可能会有适合你的EFI，如果使用 Clover ，建议参考冰水加劲Q的黑苹果安装教程

使用Disk Genius将EFI复制进U盘的EFI分区

其他设置

BIOS设置

重启进入BIOS，设置硬盘AHCI、关闭安全启动，保存并退出。

U盘启动

再次启动电脑，进入启动项选择(Dell是在显示Logo时按 F12 )

选择你的U盘

如果没有显示，可以在BIOS里面添加

安装

不出意外，你就能进入到 OpenCore 的启动界面了，这里你需要选择 Install macOS

抹盘

以上不出意外，你就进入到macOS的恢复模式了

进入磁盘工具，选择你要安装的分区，点击抹掉，格式需要选择为 APFS

安装macOS

退出磁盘工具，选择 安装macOS

之后就能进入安装了，安装过程较为漫长，请耐心等待

安装分为三个阶段

  • 第一阶段，程序把文件写入硬盘，然后会自动重启，选择选择U盘进入OpenCore，选择新出现的引导项
  • 第二阶段，程序继续进行安装，完成后会自动重启，同样需要选择U盘启动进入OpenCore
  • 第三阶段，系统安装完成，进行一些自定义设置，进入系统

合并EFI分区，实现硬盘引导

安装好系统之后，你需要脱离U盘，实现硬盘引导

工具：OpenCore Configurator

打开OpenCore Configurator，菜单栏点击图标，挂载对应的EFI分区

打开分区，将EFI文件扔到分区里面即可

完成之后，你可以拔掉U盘重启了

后续

启动默认进入Windows的，可以在BIOS里面设置 OpenCore 为第一启动项。

如果感觉字体太小，可以使用冰水加劲Q的one-key-hidpi脚本，食用方法可参考我的Dell燃7000黑苹果EFI

其他问题

OpenCore Configurator如何添加boot-args引导参数

Clover卡++++

如果你使用的是冰水的EFI，并且你安装的是 15.6 以上的系统，那很有可能是 Clover 版本太低了，你需要更新一下 Clover

我是用的是白果拆机卡，用了你的 EFI 扫描不到 Wifi

因为我使用的是 DW1560 ，所以你需要把博通的网卡驱动去掉

将图中圈中的驱动后面的 √ 去掉即可

我是用的 Intel 网卡，该怎么办？

首先，你也需要把博通网卡驱动去掉

剩余步骤请参考stevezhengshiqi更新维护的驱动内置英特尔无线网卡

使用 HeliPort 后，我怎么把系统自带的wifi图标去掉？

我睡眠电脑时间不走怎么办？

你可能需要清空CMOS

有线网卡有问题?

参考:链接

FaceTime和iMessage没法登陆?

你需要自行更换三码, 参考兵叔的公众号

macOS Big Sur Beta10 进 Recovery 灰屏怎么办?

删除 config.plist 中的 prev-lang , 然后就可以进入 Recovery 了, 安装时请选择 English
